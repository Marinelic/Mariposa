# Mariposa – Obrt za čuvanje djece.

Ovo je modernizirana web stranica za obrt za čuvanje djece **Mariposa**, izrađena s fokusom na jasnoću, toplinu i dobar UX, posebno za roditelje koji prvi put dolaze na stranicu.

## Tehnologije
- **Canva** - Logo
- **HTML5**
- **Tailwind CSS**
- **JavaScript Vanilla**
- **Resposive**
